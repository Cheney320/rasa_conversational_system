# rasa_conversational_system

## Introduction
- 计划使用Rasa搭建一个类似小爱同学的对话机器人
- 对话机器人包括四大功能：
    - 闲聊功能
    - 播放音乐功能
    - 常见知识问答功能
    - 游戏功能
